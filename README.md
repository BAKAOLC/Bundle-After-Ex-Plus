# Bundle-After-Ex-Plus

LuaSTG aex+ 游戏框架，包含：
* [LuaSTG Sub 引擎](https://github.com/Legacy-LuaSTG-Engine/LuaSTG-Sub)
* THlib
* 其他工具如 HGE 粒子效果编辑器、HGE 字体生成器等
